Weather Information System (Python)
>>Project Overview
-The Weather Information System is a console-based Python application designed to display weather information for selected Indian cities.
-This project focuses on beginner-friendly logic, clean structure, and user interaction through a menu-driven interface.
-The application allows users to:
 View weather details of a specific city
 See all supported cities
 Read general weather safety tips

>>Objectives
-To understand Python dictionaries
-To implement a menu-driven program
-To practice user input handling
-To create a clean and interactive console application

>>Technologies Used
Programming Language: Python 3
Interface: Command Line Interface (CLI)
Libraries: Built-in Python functions only (no external libraries)

>>Project Structure
Weather_Information_System/
│
├── weather_app.py

>>How to Run the Project
Ensure Python 3 is installed on your system.
Open terminal or command prompt.
Navigate to the project folder.
Run the following command:
python weather_app.py
Follow the on-screen menu instructions.

>>Application Features
Menu-driven interface
Case-insensitive city search
Weather data displayed with temperature and condition
Friendly weather safety tips
Error handling for invalid city input
